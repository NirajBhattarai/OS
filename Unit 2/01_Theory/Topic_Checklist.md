# Unit 2 Topic Checklist

Use this checklist before exam or viva.

## Process and PCB

- [ ] I can draw process state transitions correctly.
- [ ] I can explain each state (New, Ready, Running, Waiting, Terminated).
- [ ] I can list key PCB fields and their purpose.
- [ ] I can explain context switch steps in order.

## Threads and Synchronization

- [ ] I can differentiate process and thread with at least 3 points.
- [ ] I can define race condition and critical section.
- [ ] I can explain mutex and why it prevents lost updates.

## IPC

- [ ] I can explain producer-consumer bounded buffer.
- [ ] I can explain mutex + condition variable roles.
- [ ] I can justify why produced and consumed sums should match.

## Scheduling

- [ ] I can explain FCFS, SJF, Priority, Round Robin.
- [ ] I can compare waiting time and turnaround time.
- [ ] I can explain starvation and aging.

## Practical Validation

- [ ] I can run `bash scripts/validate_all.sh` in `03_Practicals_CPP`.
- [ ] I can interpret PASS/FAIL outputs topic-wise.
