# Unit 1 Topic Checklist

Use before exam, viva, or lab submission. Tick each item you can answer without notes.

---

## OS Definition and Role

- [ ] I can define an operating system in one clear sentence.
- [ ] I can explain the dual role: resource manager AND extended machine.
- [ ] I can give two examples of what an application gets from the OS.
- [ ] I can explain why application programs should not access hardware directly.

---

## Three Views of an OS

- [ ] I can explain the Application View with an example.
- [ ] I can explain the System View and name two resources it manages.
- [ ] I can explain the Implementation View and name three internal components.
- [ ] I can tell which view a developer, system admin, and OS engineer each cares about most.

---

## History and Evolution

- [ ] I can list at least four eras of OS evolution in order.
- [ ] I can explain what problem each era solved.
- [ ] I can state what year UNIX was born and why it mattered.
- [ ] I can connect OS evolution to modern cloud and mobile systems.

---

## Types of Operating Systems

- [ ] I can distinguish single-user from multi-user OS with an example each.
- [ ] I can distinguish single-tasking from multi-tasking OS.
- [ ] I can define hard RTOS and soft RTOS and give a real-world use case each.
- [ ] I can explain what an embedded OS is and name one.
- [ ] I can explain what makes a distributed OS different from a network OS.

---

## System Calls and Call Handling

- [ ] I can define system call and explain why it is needed.
- [ ] I can list five common system call categories with one example each.
- [ ] I can trace the complete flow of a system call from user code to kernel and back (9 steps).
- [ ] I can explain what a trap instruction does and why it is needed.
- [ ] I can explain the difference between user mode and kernel mode.

---

## System Programs and OS Structures

- [ ] I can list four types of system programs.
- [ ] I can explain monolithic kernel structure with pros and cons.
- [ ] I can explain layered OS structure with pros and cons.
- [ ] I can explain microkernel structure with pros and cons.
- [ ] I can compare monolithic vs microkernel for fault isolation and speed.
- [ ] I can explain what loadable kernel modules are and which OS uses them.

---

## Virtual Machines

- [ ] I can define virtual machine and hypervisor.
- [ ] I can distinguish Type 1 and Type 2 hypervisors with an example each.
- [ ] I can list four benefits of using virtual machines.
- [ ] I can explain how VMs relate to OS concepts (context switch, page tables).

---

- [ ] All five programs print `Validation: PASS`.
- [ ] I can explain the output of each program using Unit 1 theory.
